Sistema Fotovoltaico 

O sistema fotovoltaico utiliza a energia solar, no caso irradiância para produzir energia elétrica, essa energia gerada pelas placas é injetada no inversor solar. Sendo esse responsável por converter a energia gerada pelos placas fotovoltaicas.
A conversão é feita de corrente elétrica continua (CC) em corrente elétrica alternada (CA), pronta para ser conectada a equipamentos elétricos, como por exemplo geladeiras, motores, ar condicionados e outros. Ao lado temos uma illustrator que representa esse resumo, no caso essa informação é básica de como funciona o sistema fotovoltaico.

O que é necessário saber? 

Para se comprar um sistema de fotovoltaico é necessário entender o conceitos de energia, chamado de kilowatts (kWh) e potência elétrica (kW), as placas fotovoltaicas utiliza o conceito de potência elétrica (kW), e a sua fatura de energia elétrica  é dada em  Energia(kWh) .

Portanto suponto uma conta de energia de 100 (kWh), que aproximadamente são 82 reais, para abater essa conta o engenheiro responsável terá que escolher uma placa fotovoltaica, no caso cada modelo de placa possui uma potência pico (Wp), que pode variar de 350 a 500 Wp ou até mais altas, porém placas a cima de 500 Wp são mais difíceis de achar.  Portanto o nosso engenheiro resolveu utilizar uma placa de 410Wp, para este sistema hipotético. Lembrando, uma placa fotovoltaica não é melhor ou pior só porque possui uma potência  mais elevada ela simplesmente gera mais energia, porém ela é maior e mais cara. 

Porém apenas uma placa não consegue gerar energia o suficiente para abater 100kWp, é necessário adquirir um conjunto de placas 410Wp, nesse caso o conjunto de placas é chamado de painéis solares.  Para conseguimos abater 100kWp, nosso sistema deverá ter no minimo 2 placas, que em um mês irá gerar 99kWh de energia. 

Agora é necessário escolher o inversor, porém a escolha do inversor é mais complicada, irá depender de vários fator, porém o mais importante é que o inversor deve ser da mesma potência que o painéis solares, que no nosso caso possui 2 placas de 410, portanto precisamos de um inversor de potência igual ou próxima de 820Wp, para esse caso hipotipótico utilizaria um microinversor de 700Wp, e assim temos um sistema para abater uma conta de 100kWp/mês.

Tente fazer um resumo mais didático para explicar as a nuances do sistema fotovoltaico, vejo muitos  clientes que me procuram com dúvida sobre o sistema fotovoltaico, e o objetivo da SoluEnd é ser transparente e nosso foco é a sua satisfação, por tanto, antes de você adquirir um sistema fotovoltaico entre em contato com a gente, teremos o maior prazer em analisar o seu caso e explicar se o que você esta comprando realmente vale apena.

Tipo de Sistemas Fotovoltaicos

MICROINVERSOR

Microinversores são recomentado para sistemas menores ou sistemas com sombreamento. Utilizado usualmente para otimizar a geração de energia. 

INVERSOR STRING

Recomentado para sistemas residenciais ou rurais de médio porte, com pouca sombreamento. Possui um custo beneficio menor que os sistemas de microinversores.  

INVERSOR CENTRAl

São inversores para sistemas maiores, sem sombreamento e com foco na produção de energia, menor custo de instalação.

Instalação OnGrid

Instalação de sistema fotovoltaico conectado a rede, ou mais conhecido com sistema OnGrid, são atualmente os queridinhos do setor. É justamente o sistema que mante-se conectado à rede de distribuição, nesse caso, o sistema fotovoltaico gerencia a produção de energia das placas, e essa energia será injetada na rede, assim gerando os créditos de energia, que são utilizados em momentos que não há produção, durante a noite ou em dias que o consumo excede a produção.

Principais Vantagens do Sistema OnGrid: gera créditos para serem usados no futuro ou em outra unidade consumidora, assim deixando praticamente zerada a sua conta de energia, outra vantagem o sistema fotovoltaico consegue reduzir a sua demanda e não tem necessidade de uso de bateria, diminuído assim o custo de instalação.  

Precisa de ajuda ?

Principal Desvantagem do Sistema OnGrid: o sistema é dependente da rede elétrica. Devido a mecanismos de operação do inversor fotovoltaico, o sistema opera apenas se existir abastecimento de energia, caso venha ocorrer queda no fornecimento de energia o inversor deliga-se automaticamente.

Tipo de sistema fotovoltaicos OnGrid:
     - Sistema com MICROINVERSOR
     - Sistema com INVERSOR STRING
     - Sistema com INVERSOR CENTRAL

Instalação OffGrid

Instalação de sistema fotovoltaico desconectado da rede, ou mais conhecido com sistema OffGrid, são atualmente utilizados para atender locais remotos,  ou para manter um equipamento protegido da oscilação no fornecimento de energia.  
Principais Vantagens do Sistema OffGrid: autonomia na instalação, podendo ser instalado em locais remotos, funciona como gerador, protegendo o equipamento de falha por fornecimento. Sendo utilizado para sistema de bombeamento, telecomunicação, iluminação, eletrificação rural simples e entre outros. 

Precisa de ajuda ?

Principal Desvantagem do Sistema OffGrid: custo de instalação elevado, pois utiliza baterias, controlador de carga e inversores, projetado por sobre demanda, é necessário otimizar o sistema para atender a carga específica, pois quanto maior for o sistema OffGrid, maior será o investimento financeira para a sua implantação.

MICROINVERSOR

Os Microinversores são inversores de pequeno porte, especialmente desenvolvidos para sistemas menores ou sistema que necessitem de uma maior otimização e eficiência no processo de geração energia.

Os sistemas com microinversores surgiram para melhorar a geração de energia em locais que apresentem sombreamentos parciais, pois estes tipos de sistemas contam com controle de potência independente por placas, diferentemente de sistema com os inversores strings tradicionais.

Vantagens de se utilizar os microinvesores, podemos minimizar os efeitos de sombreamento, facilidade para a montagens e a substituição de equimosamentos com defeitos. Facilidade para identificar placas com mau funcionamento, possibilita utilizar placas de diferentes potências e redução do custo para a ampliação da potência do sistema.

Desvantagens do sistema com microinversores, os microinversores acabam tendo o seu custo beneficio mais elevados, portanto possui o seu playback maior. 

Inversor String

Os inversores strings são equipamentos solares, responsáveis pela conversão da energia gerada pelas placas fotovoltaicas, a conversão permite que a energia seja utilizada e injetada na rede elétrica. Os inversores strings são maiores que os Microinversores, ou seja mais robustos, consegue suportar tensões mais altas (+800V), permitindo assim a conexão de strings maiores acima de 5 módulos até 20 módulos.

Os inversores strings se diferenciam dos microinversores, devido a quantidade de MPPT no sistema, geralmente tem-se 1 MPPT para cada string, podendo ser uma string com 5 a 20 módulos, por isso o nome inversor string, devido ao fato que o controle da potência é feito na string de forma uniforme para todos os módulos. 

A vantagens de se utilizar o inversor string, é que ele reduz o valor, permitindo o barateamento do sistema fotovoltaico, funciona muito bem para instalações que possuem a mesma orientação, inclinação e configuração da potência dos módulos.

Desvantagem dos inversores strings, projeto deve ser bem projetado e executado, é necessário realizar uma analise da qualidade do sombreamento no local da instalação, pois é um sistema sensível a sombreamentos parciais e sujeiras. Não pode ser utilizados módulos diferentes na instalação e nem aumentar a quantidade de módulos na string após a implementação do sistema. 

Inversor Central

 Os inversores centrais são utilizados em sistemas de grande porte, como sistemas industriais, condomínios e usinas. O inversor central é conectado a um grande número de módulos fotovoltaicos, possibilitando a conexão de fileiras em paralelo para aumentar a corrente gerada, fazendo com que a conversão seja centralizada em um único inversor, como objetivo de simplificar a instalação e aumentar a eficiência em comparação aos inversores de menor escala anteriormente citados.

Principal vantagem é a redução de gastos, pois nesta configuração se utiliza um número reduzido de inversores. Porém tem-se a desvantagem de confiabilidade, pois se o inversor central falhar, o sistema pode ficar parado sem gerar energia. Os inversores centrais também não permitem um MPPT eficiente, pois o ponto de máxima potencia não é independente para cada string.

Bombeamento Solar

O bombeamento de água realizado através de bomba solar oferece diversas soluções para melhorar a gestão dos recursos hídricos principalmente em propriedades rurais, ou lugares que não tem acesso a rede elétrica .

A Bomba Solar permite utilizar a energia solar por meio de placas fotovoltaicas instaladas diretamente a bomba ou indiretamente, por intermédio de um Drive.

Na imagem ao lado é apresentado um dos modelos de Bomba Solar que recomendamos, é um modelo de Bomba Submersa, que é colocado dentro do poço, mas também temos Bomba Solar de Superfície.

As principais vantagens de se utiliza bombas solares:
1 - Economia na montagem do sistema de bombeamento.
2- Economia na conta de energia.
3- Autonomia de funcionamento mesmo se a energia acabar.
4- É possível utilizar baterias para operação durante a noite.
